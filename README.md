A website that helps users create optimized prompts for different AI tools in seconds.  
Instead of guessing how to phrase prompts, this platform guides users with templates, structure, and suggestions designed for modern AI models.

## 🚀 Features

- Prompt generator for multiple AI tools
- Structured prompt templates
- Easy-to-use web interface
- Customizable prompt inputs
- Copy-ready prompts
- Works for text, image, coding, and research prompts

## 🧠 Supported AI Tools

This project is designed to generate prompts for tools such as:

- ChatGPT
- Claude
- Gemini
- Midjourney
- Stable Diffusion
- Other AI platforms

## ⚙️ How It Works

1. User selects the AI tool they want to use
2. The website asks structured questions
3. Inputs are combined into an optimized prompt
4. The generated prompt can be copied and used directly in the AI tool
